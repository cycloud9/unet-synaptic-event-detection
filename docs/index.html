Fast, robust detection of synaptic events with a 1D U-Net

An end-to-end framework for detecting EPSCs and IPSCs in noisy, long-duration patch-clamp recordings, with pointwise temporal localization, overlap-add inference, and an interactive review GUI.

Method

Dense temporal prediction for event-level localization

The model predicts a continuous event-probability trace rather than assigning one label to each window. Skip connections preserve event morphology, while overlap-add inference reconstructs smooth full-length predictions.

01

Morphology-aware input

Window-level preprocessing emphasizes event shape while reducing dependence on absolute amplitude scale.

02

Pointwise output

The network returns a probability value at each time point, enabling dense localization of event centers.

03

Overlap-add inference

Overlapping windows and weighted reconstruction reduce boundary artifacts in long recordings.

04

Manual review

The GUI supports verification, correction, potential-event inspection, and CSV export.

EPSC benchmark

Higher recall without sacrificing precision

Across MSN recordings, the U-Net achieved the strongest overall F1 score among the evaluated approaches and remained sensitive to low-amplitude and temporally adjacent events.

EPSC recall0.912

Compared with 0.696 for SimplyFire, 0.679 for miniML, and 0.478 for deconvolution.

EPSC precision0.917

Maintained high precision while recovering more true events than the comparison methods.

EPSC F1 score0.913

Best overall balance between precision and recall in the benchmark set.

Performance on challenging event types

Additional analyses evaluate temporally overlapping events and small-amplitude events, two common failure modes in synaptic event detection.

IPSC extension

Dilated bottleneck for slower inhibitory-current kinetics

IPSCs have broader temporal profiles than EPSCs. A parallel dilated-convolution bottleneck expands temporal context while preserving pointwise prediction.

Generalization

Transfer to human brain organoid recordings

The EPSC-trained model was evaluated on organoid recordings to assess transfer across preparation type and recording frequency.

Software

Interactive workflow for experimental review

The desktop GUI integrates trace loading, model selection, inference, probability visualization, event navigation, manual verification, and CSV export.

  • EPSC and IPSC mode selection
  • CSV and ABF trace loading
  • Detected-event and potential-event review
  • Manual event promotion, rejection, and export
